Introduction

This course, "Financial Econometrics", is about the analysis of financial market data, mainly return data.

  • "Financial" means we analyze the financial data.
  • "Econometrics" means we apply statistical models to the data.

Our course is a data-oriented course, which means we need to do data analysis/modelling throughout the entire semester.

To understand the course, read the syllabus.

Introduction

Themes related to the course

  • Always look at the data.
  • All models are false.
  • Bias-variance tradeoff.
  • Uncertainty analysis.
  • Financial markets data are not normally distributed.
  • Variances are not constant.

Prerequisite

Probability

  • Probability space and probability measure.
  • Random Variables, vectors.
  • Probability Distribution (CDF, PDF) and some common examples.
  • Normal Distribution and its properties.
  • Moments, moment generating functions.
  • Law of Large Number.
  • Central limit theorem for i.i.d. case.

Prerequisite

Statistics

  • Data and sample distributions.
  • Discriptive statistics.
  • Estimation problems, moment estimation procedure.
  • Maximum likelihood method.
  • Mean, bias, variance, correlation, covariance and etc.
  • Point and interval estimators.
  • Asymptotic properties of estimators.
  • Hypothesis Testing and related concepts.
  • Model selection and prediction.
  • classical linear regression models.

Writing Style Guide

There are several simple principles we need to mention for the data anlaysis report writing:

  • Reproducible research/report. (vs. replication)
  • Making work into pipeline.
  • Organizing a data analysis with necessary scripts, datasets, figures, tables, and other documentation.
  • Writting the final report with clear structure.

A more detailed reference about reproducible reporting in data science is the following book by Professor Roger D. Peng:

Report Writing for Data Science in R

Structure of a Data Analysis

We state a general stucture guide for data analysis for projects in this course.

  • Define the question, raw data, processed data with transforming script and documents
  • Exploratory data analysis with evidence-based intepretation
  • Statistical modeling/prediction with statistical programming script
  • Interpretation of the results
  • Challenging the findings and synthesizing the results
  • Reproducible code and result data with necessary code book or other documents
